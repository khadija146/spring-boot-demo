# Spring Boot Demo Project

## Project Description
This project is a simple Spring Boot backend application created for a university assignment.
It demonstrates a basic REST API structure using Java, Spring Boot, and Maven.

## Technologies Used
- Java 17
- Spring Boot
- Maven
- GitHub
- Railway

## How to Run Locally
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Make sure Java 17 is selected
4. Run the Spring Boot application
5. Open the local server in the browser if needed
